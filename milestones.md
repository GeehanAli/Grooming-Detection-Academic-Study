# Milestones — MIT Emerging Talent ELO2 Capstone

This project follows a milestone structure similar to the CDSP, adapted for an academic capstone. Each milestone will also be tagged in Git history (e.g. `m1-problem`, `m2-data`, etc.).

---

## Milestone 1: Problem Identification

**Git tag (later):** `m1-problem`

**Goals:**
- Clearly define the problem.
- Write focused research questions.
- Summarize key literature on grooming and detection.
- Acknowledge major constraints (privacy, data access, cost, culture).

**Planned artifacts:**
- `/docs/milestone_1_problem_identification.md`
- Updated `project_overview.md` if needed.

---

## Milestone 2: Data Collection & Preparation

**Git tag (later):** `m2-data`

**Goals:**
- Generate synthetic grooming and non-grooming chats.
- Document how the data was created and labeled.
- Save raw and processed datasets.

**Planned artifacts:**
- `/data/raw/synthetic_chats_raw.csv`
- `/data/processed/synthetic_chats_model_ready.csv`
- `/docs/data_documentation.md`

---

## Milestone 3: Modeling & Analysis

**Git tag (later):** `m3-analysis`

**Goals:**
- Implement a simple keyword baseline.
- Train at least three ML models (e.g., Logistic Regression, SVM, MLP).
- Evaluate performance on the synthetic dataset.
- Discuss limitations (imbalanced real world vs. balanced synthetic data, false positives, etc.).

**Planned artifacts:**
- `/notebooks/03_modeling_and_evaluation.ipynb`
- `/models/` (saved models if needed)
- Summary written in `/docs/analysis_summary.md`

---

## Milestone 4: Communicating Results

**Git tag (later):** `m4-communication`

**Goals:**
- Write a clear, non-technical summary of the project and findings.
- Prepare tables/plots to compare keyword baseline vs. ML models.
- Make the work understandable to a non-technical audience.

**Planned artifacts:**
- `/docs/public_facing_summary.md`
- Figures saved in `/docs/figures/` (optional)

---

## Milestone 5: Final Presentation & Reflection

**Git tag (later):** `m5-presentation`

**Goals:**
- Create a short slide deck or video explaining the project.
- Reflect on what worked, what didn’t, and what you would do next.

**Planned artifacts:**
- `/presentation/slides.pdf` or `/presentation/slides.pptx` (or a video link)
- `/docs/final_reflection.md` (750–1000 words)
